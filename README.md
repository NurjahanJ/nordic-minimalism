# Nordic Minimalism

A comprehensive website demonstrating authentic Scandinavian design principles through research-driven implementation.

## About

This project showcases **Nordic Minimalism** (Scandinavian Design) through a carefully crafted website that embodies the core principles of calm, spacious, natural design. Built from extensive research of authoritative sources including museums, publications, and contemporary design studios.

## Key Features

- **Authentic Nordic Design System** - Warm neutrals, humanist typography, generous spacing
- **Research-Driven** - Based on comprehensive analysis of Scandinavian design principles
- **Responsive Design** - Mobile-first approach with accessibility considerations
- **Single-Page Architecture** - Clean, focused user experience

## Design Principles

- **Lagom** (Swedish): Not too much, not too little, just right
- **Hygge** (Danish): Warm, cozy, inviting atmosphere
- **Democratic Design**: Accessible, functional beauty for everyone
- **Natural Connection**: Colors and materials inspired by nature

## Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd nordic-minimalism
   ```

2. **Start local server**
   ```bash
   python -m http.server 8000
   ```

3. **View in browser**
   ```
   http://localhost:8000
   ```

## Color Palette

- **Base**: Warm white (#FAFAF9), Linen (#F5F4F0)
- **Neutrals**: Greige (#E5E3DD), Warm gray (#CCC9C1)
- **Accents**: Sage green (#8B9A8A), Warm taupe (#B4A89A)
- **Text**: Charcoal (#4A4A4A) - never pure black

## Typography

- **Font**: Inter (humanist sans-serif)
- **Body**: 18px with 1.7 line-height
- **Generous spacing** for breathing room
- **Soft hierarchy** with gentle size differences

## Research Sources

- **Museums**: Designmuseum Danmark, ArkDes Stockholm, Museum of Finnish Architecture
- **Publications**: Kinfolk Magazine, Ark Journal, Scandinavian Design (Taschen)
- **Studios**: Norm Architects, FRAMA CPH, Muuto, Note Design Studio
