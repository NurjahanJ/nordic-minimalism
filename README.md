- **Democratic Design**: Accessible, functional beauty for everyone
- **Natural Connection**: Colors and materials inspired by nature

## ğŸ¨ Design System

### Color Palette
- **Base**: Warm white (#FAFAF9), Linen (#F5F4F0)
- **Neutrals**: Greige (#E5E3DD), Warm gray (#CCC9C1)
- **Accents**: Sage green (#8B9A8A), Warm taupe (#B4A89A)
- **Text**: Charcoal (#4A4A4A) - never pure black
- **Enhanced Contrast**: Darker sage (#5A6759) for accessibility compliance

### Typography
- **Font**: Inter (humanist sans-serif)
- **Body**: 18px with 1.7 line-height
- **Generous spacing** for breathing room
- **Soft hierarchy** with gentle size differences
- **Mobile Optimized**: Responsive font scaling across all devices

### Responsive Design
- **Breakpoints**: 320px, 480px, 768px, 1024px, 1440px+
- **Mobile-First**: Progressive enhancement approach
- **Touch Targets**: 44px+ minimum for accessibility
- **Hamburger Navigation**: Full-screen mobile menu with animations
